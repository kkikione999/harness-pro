## 🚀 三步上手指南

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│               第 1 步：准备你的"工地"                        │
│                                                             │
│  在你的项目目录下创建结构：                                  │
│                                                             │
│  your-project/                                              │
│  ├── .claude/                                               │
│  │   ├── agents/                                             │
│  │   │   └── your-worker.md    ← Worker Agent 配置           │
│  │   └── skills/                                             │
│  │       ├── your-main/                                      │
│  │       │   └── SKILL.md        ← Main Agent 技能           │
│  │       └── your-worker/                                    │
│  │           └── SKILL.md        ← Worker Agent 技能          │
│  ├── .worktrees/         ← 独立工作区（就像工地的工作间）     │
│  ├── docs/                                                      │
│  │   └── tasks/           ← 任务书存储处                        │
│  └── src/                ← 主楼（最终代码所在）                │
│                                                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│               第 2 步：写第一个任务书                         │
│                                                             │
│  在 docs/tasks/ 创建任务001.md：                               │
│                                                             │
│  # Task 001: 创建基础模型                                     │
│                                                             │
│  ## Goal                                                    │
│  创建 User 类，表示用户                                       │
│                                                             │
│  ## Scope                                                    │
│  • 创建 src/user.py                                          │
│  • 包含 User 类（name, email属性）                           │
│  • 创建 tests/test_user.py                                   │
│  • 写5个测试用例                                            │
│                                                             │
│  ## Non-Goals                                                │
│  • 不做数据库存储                                            │
│  • 不做用户认证                                              │
│                                                             │
│  ## Acceptance Criteria                                      │
│  • [ ] User 类存在                                          │
│  • [ ] 有 name 和 email 属性                                │
│  • [ ] 所有测试通过                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│               第 3 步：让"工人"开始工作                      │
│                                                             │
│  在 Claude Code 中：                                          │
│                                                             │
│  1. 加载 Worker 技能                                          │
│     /load .claude/skills/your-worker/SKILL.md               │
│                                                             │
│  2. 告诉 Claude 执行任务                                      │
│     "执行 docs/tasks/task-001.md 中的任务"                    │
│                                                             │
│  3. Claude 会自动：                                           │
│     • 读取任务书                                             │
│     • 创建 worktree                                          │
│     • 写代码                                                │
│     • 添加测试                                              │

│     • 提交合并                                              │
```
