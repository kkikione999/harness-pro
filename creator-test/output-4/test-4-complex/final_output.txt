## 完成摘要

重构成功！**验证结果：build ✓, test (21/21) ✓**

### 架构变化

**之前（紧耦合）：**
```
AppWindowManager → 直接创建 AppState()
PreviewWindowController → 持有 AppState 具体类型
ContentView → @ObservedObject var appState: AppState
```

**之后（协议注入）：**
```
AppWindowManager → 持有 AppStateFactory 协议
                 → 通过工厂创建 AppState
PreviewWindowController → 持有 any AppStateProtocol
ContentView → 仍用 AppState（@ObservedObject需要具体类型）
```

### 关键变更

| 文件 | 变更 |
|------|------|
| `AppState.swift` | 新增 `AppStateProtocol` 协议 |
| `AppWindowManager.swift` | 新增 `AppStateFactory` 协议 + `DefaultAppStateFactory` 实现 |
| `PreviewWindowController` | 持有 `any AppStateProtocol` + 具体 `AppState` 引用（用于 SwiftUI 兼容性） |
| `AppStateTests.swift` | 适配 `TaskBox` 包装器用于身份比较 |

### 耦合消除
- `AppWindowManager` 不再直接依赖 `AppState` 具体类型
- 可通过注入自定义 `AppStateFactory` 实现测试替身
- `PreviewWindowController` 对外使用协议类型
